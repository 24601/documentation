:::caution[Usando con pnpm]
Debido a problemas en la parte superior de Prettier, el plugin no se detectará automáticamente cuando se use [pnpm] (https://pnpm.io/). Para que lo encuentre, se debe agregar el siguiente parámetro cuando se ejecuta Prettier:


```shell
prettier --write --plugin-search-dir=. .
```
