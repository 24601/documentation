Si está ejecutando OpenReplay detrás de un proxy, adapte su configuración para que se vea como la configuración a continuación.

## Configuración de NGINX

Asegúrese de que su proxy esté enviando los encabezados `X-Forwarded-For` y `X-Forwarded-Proto`. SSL también debe estar habilitado para que OpenReplay funcione. El resto de la configuración debe verse así:


```nginx
location / {
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_set_header Host $http_host;
        proxy_redirect off;
        proxy_pass http://1.2.3.4; # replace with OpenReplay instance IP
    }
```


## Configuración de Apache2

Asegúrese de que los módulos `proxy`, `proxy_http` y `proxy_html` estén habilitados. SSL también debe configurarse, el resto de la configuración debe ser similar a esta:


```apacheconf
<VirtualHost *:443>
    ProxyPass / http://1.2.3.4 <!-- replace with instance IP -->
    RequestHeader set X-Forwarded-Proto expr=%{REQUEST_SCHEME}
    # SSL + other config here
</VirtualHost>
```
