OpenReplay API permite administrar datos de usuario y exportar eventos capturados para análisis externo.

## HTTP API
- [Projects](/api/projects): Para la creación de nuevos proyectos.
- [Sessions](/api/sessions): Para recuperar todas las sesiones de un usuario específico.
- [Events](/api/events): Para extraer todos los eventos de una grabación en particular.
- [Users](/api/users): Para controlar y borrar datos de usuario.
- [Jobs](/api/jobs): Para administrar trabajos de eliminación de datos de usuario.
- [Assist](/api/assist): Para recuperar sesiones en vivo.
