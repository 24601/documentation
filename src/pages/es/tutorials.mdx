Sigue los tutoriales para aprovechar al máximo OpenReplay.

## Cómo se hace

Aprende con los siguentes tutoriales sobre casos reales de usos de OpenReplay.

### Complementos
- [Crea tus propios plugins](/tutorials/build-plugins): Aprenda a crear sus propios plugins personalizados en su instancia OpenReplay autoalojada.
- [Asistencia](/tutorials/assist): Ofrezca a sus usuarios finales la mejor asistencia y experiencia de incorporación mediante la reproducción en directo de las sesiones.
- [Complemento Fetch](/tutorials/fetch): Aprenda a configurar y desinfectar la información de las solicitudes con el complemento Fetch.
- [Complemento Axios](/tutorials/axios): Aprenda a configurar y desinfectar la información de las solicitudes utilizando el plugin Axios.
- [Complemento Redux](/tutorials/redux): Aprende a configurar el plugin Redux para capturar cambios de estado en tus repeticiones de sesión.
- [Complemento VueX](/tutorials/vuex): Aprende a configurar el plugin VueX para capturar cambios de estado en tus repeticiones de sesión.
- [Complemento GraphQL](/tutorials/graphql): Aprenda a configurar el plugin GraphQL para capturar datos de operaciones en sus repeticiones de sesión.
- [Complemento Zustand](/tutorials/zustand): Seguimiento de los datos de estado de Zustand con el plugin Zustand Tracker.
- [Complemento Pinia](/tutorials/pinia): Seguimiento de los datos de estado de Pinia con el plugin Pinia Tracker.

### Características

- [OmniSearch](/tutorials/omnisearch): Aproveche nuestra potente función de búsqueda para encontrar el tipo de sesión adecuado para su proyecto.
- [Agregar usuarios a tu equipo](/tutorials/adding-users): Aprende a añadir usuarios a tu instancia autoalojada de OpenReplay.
- [Tipo de problemas](/tutorials/issues): Aprende qué significan todos los problemas autodetectados y cómo sacarles el máximo partido.
- [Aumenta la abilidad de seguimiento con Eventss personalizados](/tutorials/custom-events): Obtenga información sobre los eventos personalizados y cómo puede aumentar sus superpoderes de seguimiento.
- Dashboards:
   - [Crea dashboards personalizados](/tutorials/custom-dashboard): Aprenda a aprovechar los cuadros de mando personalizados para visualizar el estado actual de su sistema.
   - [Crea embudos personalizados](/tutorials/funnels): Aprenda a crear embudos personalizados para entender qué está causando sus caídas de conversión.
   - [Mostrar errores de JavaScript en un cuadro de mandos personalizado](/tutorials/js-errors-dashboard): Añade una métrica personalizada (Errores JS) en tus Dashboards con este tutorial.
