Tu instancia OpenReplay puede ser configurada fácilmente. Aquí puedes encontrar los tutoriales para habilitar los servicios de email, proteger tu instalación y eliminar grabaciones de la base de datos

## Step-by-step guides
- [Configure SMTP](/configuration/configure-smtp): Configura un SMTP para recibir alertas, reportes semanales, invitar nuevos usuarios a OpenReplay y resetear contraseñas.
- [Secure OpenReplay](/configuration/secure-or): Protege OpenReplay configurando un SSL, reCaptcha y configurando un CSP.
- [Proxy Settings](/configuration/proxy-settings): Ejecuta OpenReplay utilizando un proxy como NGINX o Apache2.
- [Cleanup Storage](/configuration/cleanup-storage): Cómo las grabaciones son almacenadas, borrar en bloque desde la base de datos y limpiar el almacenamiento.
- [External Recordings' Storage](/configuration/external-storage): Cambia el almacenamiento predeterminado para tus grabaciones.
- [External Database](/configuration/external-db): Usa una base de datos Postgres externa para OpenReplay.
- [Single-Sign-On (SAML)](/configuration/sso): Configurar el inicio de sesión único (SSO) a través de SAML.
