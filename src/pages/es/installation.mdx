Esta sección contiene todo lo que necesitas para iniciar OpenReplay, desde funciones básicas hasta systemas más avanzados.

## Cómo se hace
- [Cómo identificar a un usuario](/installation/identify-user): Asocie su ID de usuario interno a la sesión que se está grabando.
- [Cómo agregar usuarios a tu equipo](/tutorials/adding-users): Aprenda a añadir usuarios a tu equipo.
- [Cómo enviar Metadata](/installation/metadata): También denominados rasgos o variables de usuario, los metadatos proporcionan información adicional sobre los usuarios mientras se graban las sesiones.
- [Cómo enviar Eventos personalizados](/installation/custom-events): Añada más inteligencia registrando eventos específicos del dominio junto con las repeticiones de sesión.
- [Cómo hacer un informe de errores](/installation/error-reporting): Notificar errores y excepciones y verlos en la pestaña 'Errors'.
- [Cómo agregar Source-Maps](/installation/upload-sourcemaps): Facilitar la depuración viendo el contexto del código fuente obtenido de las trazas de pila en su forma original.
- [Cómo sanitizar los Datos](/installation/sanitize-data): Cómo desinfectar los datos a nivel global y/o granular.
- [Cómo crear una heuristica personalizada](/tutorials/custom-heuristics): Aprenda a añadir lógica personalizada al backend para comprobar el comportamiento de uso personalizado.
- [Cómo utilizar la barra OmniSearch](/tutorials/omnisearch): Aprenda a sacar el máximo partido a nuestra potente barra de búsqueda.
- [Cómo crear tu propio Dashboard personalizado](/tutorials/custom-dashboard): Cree sus propias métricas y cuadros de mando con este tutorial.
