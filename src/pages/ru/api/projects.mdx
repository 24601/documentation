## Создание проекта

Создайте новый проект, указав его `имя`.

### Метод
`POST`

### URL
`/api/v1/projects`

### Параметры

| Имя | Тип | Описание |
|----------|-------------|-------------|
| имя | строка | Имя проекта, который вы создаете (например, MyApp, Production, Staging) (обязательно) |

### Заголовки запроса

`Авторизация: {КЛЮЧ_API_ВАШЕЙ_ОРГАНИЗАЦИИ}`: Ключ можно найти в «Настройках»> «Аккаунт»> «Ключ API организации».

### Коды состояния

`200`: Ответ отправлен в формате JSON в теле

### Пример запроса


```shell
curl -X POST \
  https://openreplay.example.com/api/v1/projects/MyFirstProject \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
  --data-raw '{
    "name": "MyFirstProject"
}'
```


### Пример ответа


```json
{
  "data": {
    "projectKey": "QsLA07Oa5sqTfwnNaFwS",
    "name": "MyFirstProject",
    "gdpr": {
      "maskEmails": true,
      "sampleRate": 50,
      "maskNumbers": true,
      "defaultInputMode": "plain"
    }
  }
}
```


## Получение проекта

Возвращает детали указанного проекта.

### Метод
`GET`

### URL
`/api/v1/projects/:projectKey`

### Параметры

| Имя | Тип | Описание |
|----------|-------------|-------------|
| projectKey | строка | ID проекта, который вы отслеживаете (обязательно) |

### Заголовки запроса

`Авторизация: {КЛЮЧ_API_ВАШЕЙ_ОРГАНИЗАЦИИ}`: Ключ можно найти в «Настройках»> «Аккаунт»> «Ключ API организации».

### Коды состояния

`200`: Ответ отправлен в формате JSON в теле

### Пример запроса


```shell
curl -X GET \
  https://openreplay.example.com/api/v1/projects/7ePSXFuQVidq9pqS6Xyn \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
```


### Пример ответа


```json
{
  "data": {
    "projectKey": "7ePSXFuQVidq9pqS6Xyn",
    "name": "MyFirstProject"
  }
}
```

### Заголовки запроса

### Коды состояния

`200`: Ответ отправлен в формате JSON в теле

### Пример запроса


```shell
curl -X GET \
  https://openreplay.example.com/api/v1/projects \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
```


### Пример ответа


```json
{
  "data": {
    "projectKey": "7ePSXFuQVidq9pqS6Xyn",
    "name": "MyFirstProject"
  },
  {
    "projectKey": "9cACFRuQVidq0sqS7Bny",
    "name": "MySecondProject"
  }
}
```
