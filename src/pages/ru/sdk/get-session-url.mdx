Этот метод позволяет получить URL активной сессии.
URL предназначен для панели управления OpenReplay.

Этот метод полезен, если вы хотите интегрировать OpenReplay в свой стек.

## Подпись


```js
getSessionURL(withCurrentTime?: boolean): string | undefined
```
