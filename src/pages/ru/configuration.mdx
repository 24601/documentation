Ваш экземпляр OpenReplay можно легко настроить. Вот пошаговые руководства по включению почтовых служб, обеспечению безопасности установки и удалению записей из базы данных.

## Step-by-step guides
- [Configure SMTP](/configuration/configure-smtp): Настройте SMTP для получения предупреждений, еженедельных отчетов, приглашения новых пользователей в OpenReplay и сброса паролей.
- [Secure OpenReplay](/configuration/secure-or): Защитите OpenReplay, настроив SSL, reCaptcha и установив CSP.
- [Proxy Settings](/configuration/proxy-settings): Запустите OpenReplay за прокси-сервером, таким как NGINX или Apache2.
- [Cleanup Storage](/configuration/cleanup-storage): Как хранятся записи и как массово удалять их из базы данных и очищать хранилище.
- [External Recordings' Storage](/configuration/external-storage): Измените место хранения ваших записей по умолчанию.
- [External Database](/configuration/external-db): Используйте внешнюю БД Postgres для OpenReplay.
- [Single-Sign-On (SAML)](/configuration/sso): Настройте единый вход (SSO) через SAML.
