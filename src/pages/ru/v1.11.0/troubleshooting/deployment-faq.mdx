## Моя установка внезапно перестала работать
  
Может быть несколько причин. Вот как вы можете отладить ситуацию:

1. Проверьте использование диска в виртуальной машине OpenReplay. Если оно превышает 80%, службы не будут работать.

## Обновление не прошло без ошибок
  
### Я вижу «helm или другая операция (установка / обновление / откат) в процессе»

Обычно это означает, что вы повторно попробовали операцию установки / обновления. Для решения ситуации выполните нижеуказанную команду:

### Я не вижу ошибок, просто «установка не удалась»


```bash  
df -h
```

```bash
helm rollback -n app openreplay
```
