Этот метод позволяет установить идентификатор пользователя для текущей активной сессии. Это позволяет связать пользователя с повторным воспроизведением, что позволит вам фильтровать и искать сессии для одного конкретного пользователя.

## Подпись

```js
setUserID(id: string): void
```
