## Очистка данных повтора

Вы можете очистить то, что захватывает OpenReplay, как на глобальном, так и на детальном уровне. Любые замаскированные или игнорируемые данные не будут захвачены на источнике. Другими словами, эти данные никогда не покинут браузер пользователя.

<Aside type="tip">
Если вы хотите посмотреть быстрое обучение по очистке данных в ваших повторах, используйте [это видео на YouTube] (https://youtu.be/22-UCoocy6s), иначе продолжайте читать для получения дополнительной информации.
</Aside>

### Глобальный уровень

Данные могут быть очищены на уровне трекера при настройке OpenReplay.

#### Я использую OpenReplay Script

![Конфигурация проекта в OpenReplay] (/static/gdpr-1.png#center)

1) Нажмите на *Код отслеживания* вашего проекта в разделе «Настройки> Проекты».
2) Опции записи данных должны появиться прямо перед фрагментом кода:
- **Входы**: для игнорирования, замаскирования или записи всех значений ввода пользователей.
- **Не записывать никакие цифровые тексты**: для игнорирования цифровых значений, содержащихся в текстовых элементах.
- **Не записывать адреса электронной почты**: по умолчанию все адреса электронной почты, содержащиеся в текстовых элементах, замаскированы и не будут отображаться в повторах. Вы можете отключить это, отменив флажок.
3) Изменение опций записи данных влияет на фрагмент кода. Убедитесь, что скопировали и вставили обновленный скрипт в ваше веб-приложение.

#### Я использую OpenReplay NPM

Существует набор параметров, связанных с конфиденциальностью, которые вы можете передать конструктору:
- `obscureTextEmails?: boolean` Замаскировать адреса электронной почты в текстовых элементах. Адреса электронной почты будут преобразованы в случайную цепочку звездочек. По умолчанию: истина.
- `obscureTextNumbers?: boolean` Замаскировать цифры в текстовых элементах. Числа будут преобразованы в случайную це
```HTML
<!--Obscuring input fields-->
<form action="/action_page.php">
  Name: <input type="text" name="fname" data-openreplay-obscured><br>
  <input type="submit" value="Submit">
</form>
<!--Obscuring a div element-->
<div style="background-color:lightblue" data-openreplay-obscured>
  <h3>This is a sensitive information</h3>
  <p>This is an important paragraph</p>
  <img src="important_image.jpg">
</div>
```
