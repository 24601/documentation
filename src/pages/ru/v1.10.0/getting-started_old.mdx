:::предостережение[Использование с pnpm]
Из-за проблем в верхнем уровне внутри Prettier плагин не будет автоматически обнаружен при использовании [pnpm] (https://pnpm.io/). Чтобы сделать так, чтобы он нашел плагин, следующий параметр необходимо добавить при запуске Prettier:


```shell
prettier --write --plugin-search-dir=. .
```
