Получить работу трекера в приложении React довольно просто.

## Добавление кода отслеживания

Хотя вы можете [получить версию кода для копирования и вставки непосредственно из платформы при создании нового проекта](/using-or/snippet#getting-the-snippet), этот код слишком простой и затруднит настройку и добавление дополнительных плагинов.

Предполагая, что вы создаете SPA (приложение одностраничного дизайна), все, что вам нужно сделать, это добавить следующий код в вашу точку входа:


```jsx
import Tracker from '@openreplay/tracker';

const tracker = new Tracker({
  projectKey: "<your project key>",  
});

function YourMainComponent({props}) {
    //some code here...

    useEffect(() => {
        tracker.start();
    }, [])

    return //....
}
```
