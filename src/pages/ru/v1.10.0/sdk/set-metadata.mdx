Импортируйте Aside из «~/components/Aside.astro».

С помощью этого метода вы можете установить значение одного из полей метаданных, настроенных на платформе OpenReplay.

<Aside type="note">
Вы можете вызывать этот метод столько раз, сколько пожелаете, просто помните, что если вы используете один и тот же атрибут «key», то последнее значение, которое вы передали, будет сохранено на платформе.
Все предыдущие значения будут потеряны.
</Aside>

См. [метаданные] (/installation/metadata) для получения дополнительной информации.

## Подпись

```js
setMetadata(key: string, value: string): void
```
