## Инструкции

Изучите приведенные ниже учебные материалы на основе реальных примеров использования.

### Плагины
- [Создание собственных плагинов](/tutorials/build-plugins): Изучите, как создать собственные пользовательские плагины в вашем самоуправляемом экземпляре OpenReplay.
- [Assist](/tutorials/assist): Предложите лучшую поддержку и введение в работу вашим конечным пользователям с помощью записи сеансов в реальном времени.
- [Плагин Fetch](/tutorials/fetch): Изучите, как настроить и очистить информацию о запросах с использованием плагина Fetch.
- [Плагин Axios](/tutorials/axios): Изучите, как настроить и очистить информацию о запросах с использованием плагина Axios.
- [Плагин Redux](/tutorials/redux): Изучите, как настроить плагин Redux для захвата изменений состояния ваших записей сеансов.
- [Плагин VueX](/tutorials/vuex): Изучите, как настроить плагин VueX для захвата изменений состояния ваших записей сеансов.
- [Плагин GraphQL](/tutorials/graphql): Изучите, как настроить плагин GraphQL для захвата данных операций ваших записей сеансов.
- [Плагин Zustand](/tutorials/zustand): Отслеживание данных состояния из Zustand с помощью плагина Zustand Tracker.
- [Плагин Pinia](/tutorials/pinia): Отслеживание данных состояния из Pinia с помощью плагина Pinia Tracker.

### Особенности

- [OmniSearch](/tutorials/omnisearch): Воспользуйтесь нашей мощной функцией поиска, чтобы найти подходящий тип сессии для вашего проекта.
- [Добавление пользователей в вашу команду](/tutorials/adding-users): Узнайте, как добавлять пользователей в вашу собственную версию OpenReplay.
- [Типы проблем](/tutorials/issues): Узнайте, что означают все автоматически обнаруженные проблемы и как извлечь максимум из них.
- [Расширьте свою возможность отслеживания с помощью пользовательских событий](/tutorials/custom-events): Узнайте о пользовательских событиях и о том, как увеличить свои возможности по отслеживанию.
- Панели управления:
   - [Создание пользовательских панелей управления](/tutorials/custom-dashboard): Узнайте, как использовать пользовательские панели управления для визуализации текущего состояния вашей системы.
   - [Создание пользовательских воронок](/tutorials/funnels): Узнайте, как создавать пользовательские воронки, чтобы понять, что вызывает снижение конверсии.
   - [Отображение ошибок JavaScript на пользовательских панелях управления](/tutorials/js-errors-dashboard): Добавьте пользовательскую метрику (JS Errors) на ваши панели управления с помощью этого учебника.
