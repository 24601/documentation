OpenReplay API позволяет управлять пользовательскими данными и экспортировать захваченные события для внешнего анализа.

## HTTP API
- [Projects](/api/projects): Для создания новых проектов.
- [Sessions](/api/sessions): Для получения всех сеансов для указанного пользователя.
- [Events](/api/events): Для извлечения всех событий конкретной записи.
- [Users](/api/users): Для контроля и удаления пользовательских данных.
- [Jobs](/api/jobs): Для управления заданиями на удаление пользовательских данных.
- [Assist](/api/assist): Для получения живых сессий.
