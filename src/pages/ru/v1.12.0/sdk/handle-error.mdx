Если информация, отслеживаемая нашим трекером по умолчанию при возникновении ошибки JS, недостаточна, вы можете использовать этот метод.

Метод `handleError` отчитывается об отловленных исключениях, отклоненных обещаниях или ошибочных событиях вручную.

См. [отчет об ошибках] (/installation/error-reporting#manuallyloggingexceptions) для получения дополнительной информации.

## Подпись

```js 
handleError: (e: Error | ErrorEvent | PromiseRejectionEvent, metadata?: Record<string, any>) => void
```
