Cette méthode vous permet d'obtenir l'URL de la session active.
L'URL est pour le tableau de bord OpenReplay.

Cette méthode est utile si vous souhaitez intégrer OpenReplay à votre propre pile.

## Signature


```js
getSessionURL(withCurrentTime?: boolean): string | undefined
```
