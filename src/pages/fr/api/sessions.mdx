## Récupérer les sessions d'utilisateur

Renvoie la liste des sessions pour un utilisateur particulier, qui a été identifié via la fonction JavaScript `tracker.setUserID`.

### Méthode
`GET`

### URL
`/api/v1/:projectKey/users/:userId/sessions`

### Paramètres

| Nom | Type | Description |
|----------|-------------|-------------|
| projectKey | chaîne | L'ID du projet que vous suivez (requis) |
| userId | chaîne | L'ID de votre utilisateur, défini via `tracker.setUserID` (requis) |
| start_date | époque (ms) | Date de début à partir de laquelle filtrer les sessions |
| end_date | époque (ms) | Date de fin jusqu'à laquelle filtrer les sessions |

### En-têtes de demande

`Authorization: {YOUR_ORGANIZATION_API_KEY}`: La clé peut être trouvée dans 'Préférences' > 'Compte' > 'Clé API de l'organisation'.

### Codes de statut

`200`: Réponse envoyée sous forme de JSON dans le corps

### Exemple de demande


```shell
curl -X GET \
  https://openreplay.example.com/api/v1/3sWXSsqHgSKnE87YkNJK/users/mickael@openreplay.com/sessions \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
```
