## Récupérer des événements

Renvoie la liste des événements tels que les clics, les visites de page ou les entrées contenus dans une session particulière. Ce sont des événements de haut niveau (indexés) qui peuvent être recherchés dans l'application OpenReplay.

### Méthode
`GET`

### URL
`/api/v1/:projectKey/sessions/:sessionId/events`

### Paramètres

| Nom | Type | Description |
|----------|-------------|-------------|
| projectKey | chaîne | L'ID du projet que vous suivez (requis) |
| sessionId | chaîne | L'ID de la session à partir de laquelle vous récupérez les événements (requis) |

### En-têtes de demande

`Authorization: {YOUR_ORGANIZATION_API_KEY}`: La clé peut être trouvée dans 'Préférences' > 'Compte' > 'Clé API de l'organisation'.

### Codes de statut

`200`: Réponse envoyée sous forme de JSON dans le corps

### Exemple de demande


```shell
curl -X GET \
  https://openreplay.example.com/api/v1/3sWXSsqHgSKnE87YkNJK/sessions/4813023023591169/events \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
```
