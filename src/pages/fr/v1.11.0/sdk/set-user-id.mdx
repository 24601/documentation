Cette méthode vous permet de définir un ID d'utilisateur pour la session active. Cela vous permet d'associer un utilisateur à la relecture, ce qui vous permet de filtrer et de rechercher des sessions pour un utilisateur particulier.

## Signature

```js
setUserID(id: string): void
```
