:::caution[Utilisation avec pnpm]
En raison de problèmes en amont dans Prettier, le plugin ne sera pas automatiquement détecté lors de l'utilisation de [pnpm] (https://pnpm.io/). Afin de le faire trouver le plugin, le paramètre suivant doit être ajouté lors de l'exécution de Prettier :


```shell
prettier --write --plugin-search-dir=. .
```
