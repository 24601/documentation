## استرداد الأحداث

يقوم بإرجاع قائمة الأحداث مثل النقرات وزيارات الصفحات أو الإدخالات الموجودة في جلسة معينة. هذه هي الأحداث المتوسطة المفهرسة التي يمكن البحث فيها في تطبيق OpenReplay.

### الطريقة
`GET`

### الرابط
`/api/v1/:projectKey/sessions/:sessionId/events`

### المعلمات

| اسم | نوع | الوصف |
|----------|-------------|-------------|
| projectKey | string | مفتاح المشروع الذي تتبعه (مطلوب) |
| sessionId | string | مفتاح الجلسة التي تستخدمها لسحب الأحداث (مطلوب) |

### رؤوس الطلب

`Authorization: {YOUR_ORGANIZATION_API_KEY}`: يمكن العثور على المفتاح في 'التفضيلات' > 'الحساب' > 'مفتاح API للمنظمة'.

### أكواد الحالة

`200`: الاستجابة المرسلة كبيان JSON في الجسم

### طلب مثال


```shell
curl -X GET \
  https://openreplay.example.com/api/v1/3sWXSsqHgSKnE87YkNJK/sessions/4813023023591169/events \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
```
