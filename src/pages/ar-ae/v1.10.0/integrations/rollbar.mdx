كيفية التكامل مع Rollbar ورؤية الأخطاء الخلفية مع تشغيلات الجلسة.

## 1. إنشاء توكينات الوصول إلى المشروع

1. سجل الدخول إلى حساب Rollbar الخاص بك.
2. اختر مشروعك من القائمة المنسدلة (أعلى اليسار).
3. انتقل إلى *الإعدادات> المشروع * توكينات الوصول.
4. انقر فوق *إنشاء توكينات وصول جديدة*.

![Rollbar Project Access Tokens](/static/rollbar-1.png#center)

5. في النطاق اختر `read`؛ في الاسم وضع `openreplay`؛ واترك *Rate Limit* على الافتراضي.
6. انسخ توكين الوصول الجديد.

![Rollbar Active Token](/static/rollbar-2.png#center)

## 2. تمكين Rollbar في OpenReplay

الصق توكين الوصول الخاص بك في لوحة التحكم OpenReplay تحت 'التفضيلات> التكامل'.

![Rollbar Integration in OpenReplay](/static/rollbar-3.png#center)


## 3. الانتشار openReplaySessionToken

لربط حدث Rollbar مع جلسة المستخدم المسجلة، يجب تدفق توكين فريد من نوعه من الواجهة الأمامية إلى الخلفية في كل طلب تريد تتبعه. يمكن القيام بذلك باستخدام رأس HTTP مخصص. في المثال أدناه، نستخدم وظيفة `fetch` لإرسال ذلك الرأس.


```javascript
const headers = {
  Accept: 'application/json',
  'Content-Type': 'application/json',
};
if (tracker.getSessionToken()) { // or window.OpenReplay instead of tracker if you're using the snippet
  headers['X-OpenReplay-SessionToken'] = tracker.getSessionToken(); // Inject openReplaySessionToken
}
fetch('www.your-backend.com', {
  'GET',
  headers,
});
```
