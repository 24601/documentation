إذا لم يكن المعلومات الافتراضية التي يتبعها متعقبنا عند وقوع خطأ JS كافية، يمكنك استخدام هذه الدالة.

تستخدم دالة `handleError` لإبلاغ عن الاستثناءات الملاحظة، والوعود المرفوضة أو الأحداث الخاطئة يدويًا.

انظر [الإبلاغ عن الأخطاء](/installation/error-reporting#manuallyloggingexceptions) لمزيد من التفاصيل.

## التوقيع

```js 
handleError: (e: Error | ErrorEvent | PromiseRejectionEvent, metadata?: Record<string, any>) => void
```
